# マニュアル

## 命令セット一覧

| オペコード | オペランド |                      |                       | 意味                                                                                      |
| ---------- | ---------- | -------------------- | --------------------- | ----------------------------------------------------------------------------------------- |
| PUSH       | int num    |                      |                       | スタックの一番上に値 num を積む                                                           |
| POP        |            |                      |                       | スタックの一番上の要素を削除する                                                          |
| SETLOCAL   | string srt | int num              |                       | 変数名 str に値 num を保存する                                                            |
| GETLOCAL   | string str |                      |                       | 変数名 str の値をスタックの一番上に積む                                                   |
| ALLOCARR   | string str | unsigned int num     |                       | サイズ num の配列 str の領域を確保する                                                    |
| SETARR     | string str | int idx (string var) | int num (string var2) | 配列 str のインデックス idx(var) 番目に値 num(var2) を保存する                            |
| GETARR     | string str | int idx (string var) |                       | 配列 str のインデックス idx(var) 番目の値をスタックの一番上に積む                         |
| FREEARR    | string str |                      |                       | 配列 str の領域を解放する                                                                 |
| FUNC       |            |                      |                       |                                                                                           |
| CALL       |            |                      |                       |                                                                                           |
| RET        |            |                      |                       |                                                                                           |
| ADD        |            |                      |                       | スタックの上二つの要素を足してスタックの一番上に積む                                      |
| SUB        |            |                      |                       | スタックの上から(一番目) - (二番目)の値をスタックの一番上に積む                           |
| MUL        |            |                      |                       | スタックの上二つの要素を乗算してスタックの一番上に積む                                    |
| DIV        |            |                      |                       | スタックの上から(二番目の) / (一番目) をしてスタックの一番上に積む                        |
| PRINT      |            |                      |                       | スタックの一番上の値を標準出力する                                                        |
| JUMP       | string str |                      |                       | ラベル str にジャンプする                                                                 |
| JPEQ0      | string str |                      |                       | スタックの一番上が0であったらラベル str にジャンプする                                    |
| GT         |            |                      |                       | スタックの上から (一番目) > (二番目) なら 1 を、そうでないなら 0 をスタックの一番上に積む |
| LT         |            |                      |                       | スタックの上から (一番目) < (二番目) なら 1 を、そうでないなら 0 をスタックの一番上に積む |
| LOGNOT     |            |                      |                       | スタックの一番上の論理否定値を、スタックの一番上に積む                                    |
| END        |            |                      |                       | プログラムの終了を宣言する                                                                |
